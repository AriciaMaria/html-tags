# HTML Fundamentos

Repositório criado para estudar e praticar os principais elementos e tags do HTML.

O projeto consiste em um quiz interativo desenvolvido apenas com HTML, com o objetivo de compreender a estrutura de uma página web e o funcionamento das principais tags sem utilizar CSS ou JavaScript.

## Objetivos

- Praticar a estrutura básica de um documento HTML.
- Aprender a utilizar elementos semânticos.
- Explorar diferentes tipos de formulários.
- Utilizar tabelas, listas, imagens e links.
- Criar uma base para estudos futuros de CSS e JavaScript.

## Conteúdos praticados

- Estrutura básica do HTML (`<!DOCTYPE>`, `html`, `head`, `body`)
- Tags semânticas (`header`, `main`, `section`, `footer`)
- Títulos e parágrafos
- Formação de texto (`strong`, `em`, `br`, `abbr`)
- Listas ordenadas e não ordenadas
- Formulários
 - `text`
 - `radio`
 - `checkbox`
 - `password`
 - `date`
 - `file`
 - `email`
 - `textarea`
 - `select`
- Tabelas (`table`, `thead`, `tbody`, `tfoot`)
- Imagens (`img`, `figure`, `figcaption`)
- Links (`a`)
- Elementos interativos (`details`, `summary`)
- Agrupamento de formulários (`fieldset`, `legend`)
